# matrip app sem backend LOGIN
Instale o node.js em: https://nodejs.org/en/;
Instale o expo no pc com o node executando "npm i -g expo" no terminal;
Baixar o expo go no celular ou no emulador de celular no pc;
Abrir a pasta do "matrip app sem backend" no terminal e depois executar "npm install expo@latest" e após "npx expo install --fix" para prevenir erros;
No terminal na pasta do projeto pelo VS CODE ou terminal do pc, executar "npm install" para instalar as dependências necessárias.
Executar "npx expo start" no terminal do VS CODE (caminho ideal) para iniciar o expo e executar o projeto
Conectar pelo qr code ou endereço "exp//:" com celular ou emulador de Android/IOS pelo pc;
